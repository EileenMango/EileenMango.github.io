ng-bind

ng-bind 是单项绑定，会将标签中原来的所有内容替换成 绑定的数据 类似（innerText的操作）

ngSanitize ng-bind-html

ngSanitize 提供了可以将html代码渲染成dom元素的指令 ng-bing-html

使用方式

  1. 引入文件 angular-sanitize.js
  2. 作为模板的依赖项，引入到模块中
  3. 直接使用ng-bind-html

由ngSanitize模块支持的一个指令，可以用来绑定数据中包含html标签的内容，会将html标签

常见的指令

ng-hide\ng-show

都是布尔值，控制元素的显示和隐藏

ng-if

如果是false，就将元素直接删除

ng-switch ng-switch-when

只有满足ng-switch-when 条件的才会出现，不满足就删除

常见的事件

ng-submit、ng-blur、ng-change、ng-dblclick

ng-disabled

ng-checkbox //单项绑定

过滤器的使用

【官方文档】(https://docs.angularjs.org/api/ng/filter)

filter 过滤器的使用

filter过滤器是用来过滤数组的，所以只能用在数组上

过滤器不仅可以通过指令在视图中调用，也可以通过js代码调用
