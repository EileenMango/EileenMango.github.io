2D转换 （tranform）

tranform可以实现元素的位移、旋转、变形、缩放

2D转换-缩放scale

  1. 参数一 ：整体缩放的倍数 如果比1大 便是放大，如果比1小 遍是缩小
  2. 参数二 ：第一个参数是x轴的缩放比， 第二个参数是y轴的缩放比

2D转换-旋转rotate

  1. 参数一：旋转的角度，单位是deg; 正角度是 顺时针旋转，负角度是 逆时针旋转

2D旋转-倾斜skew

  1. 参数一：x轴方向的倾斜 角度是正数 是逆时针倾斜；角度是负数 是顺时针倾斜
  2. 参数二：y轴方向的倾斜 角度是正数 是顺时针倾斜；角度是负数 是逆时针倾斜

2D转换-原点origin

3D转换

透视 perspective 3D效果 通过透视呈现 只能在平面上，产生近大远小的效果 视觉上的效果 不是真正意义上的3D呈现

如果想在网页上呈现3D效果 需要添加 transform-style：preserve-3d；

perspective两种写法：

  1. 在父元素中写 perspective：100px;（子元素都生效）
  2. 在子元素中写 transform perspective（100px）rotateX(90deg)

translate3D(x,y,z)

translateX/Y/Z(w px) w正数是放大 w负数是缩小

六个属性：

1. 绕X轴旋转 rotateX， 3D转换-旋转rotateX.html
2. 绕Y轴旋转 rotateY， 3D转换-旋转rotateY.html
3. 绕Z轴旋转 rotateZ，3D转换-旋转rotateZ.html
4. 在X轴移动， 3D转换-移动translateX.html
5. 在Y轴移动， 3D转换-移动translateY.html
6. 在Z轴移动， 3D转换-移动translateZ.html

补充：3D呈现 transform-style:preserve-3d;

动画 animation

通过设置多个节点来精确控制一个或一组动画，常用来实现复杂的动画效果

必要元素

1. 通过@keyframes指定动画序列； //前缀兼容 @-webkit-keyframes
2. 通过百分比将动画序列分割成多个节点；（确定执行时间）
3. 在各个节点转中分别定义各属性；
4. 通过animation将动画应用于相应元素；

@keyframes 是关键字 用来定义动画系列 后面是自定义的动画系列的名称

关键属性：

1. animation-name设置动画序列名称
2. animation-duration动画持续时间
3. animation-delay动画延时时间
4. animation-timing-function动画执行速度，linear、ease等 //：steps（3）分多少步执行完成
5. animation-play-state动画播放状态，running（运转，播放）、paused（暂停）等
6. animation-direction动画逆播，alternate（交替，轮流）等（也算在播放次数内）
7. animation-fill-mode动画执行完毕后状态，forwards（保持）、backwards（默认 还原）等
8. animation-iteration-count：2动画执行次数，infinite（无穷）等

补充：参数值的顺序 除了名字、动画时间、延时有严格顺序 其他都随意

伸缩布局 fles布局

2. 子容器 （使用伸缩的一些属性） flex：等份；

display：flex //默认是所有的元素在一行显示 默认的排列方式是横向排列

flex-wrap：wrap； 换行 (默认是 nowrap：wrap)

flew：1；设置 子容器 占剩余的宽度的多少份 //把剩余的宽度分成多少份 子容器只要设置了flex计算所有flex的和 就是总份数

flex-direction：column 控制伸缩容器的排列方向 默认是横向row纵向；

字体 webfont

web字体

1.使用某一个字体的时候，在客户的系统上面，可能出现不支持的情况，使用系统默认的字体
2.使用的字体， 是会有约束文件的（指定某一个字符对应的图）称为 字体文件
3.如果系统里没有这种文件约束，使用的是web，是需要传从服务端索取，索取字体文件 解决客户的系统抵挡不支持的字体
4.第三方的字体 可以解决不支持字体的系统
5.a/需要下载字体的文件(.eot .svg .ttf .woff .woff2 .otf)不同的字体文件的格式 用来解决不同浏览器兼容；
    b/定义第三方字体 关键字@font-face{
        c/指定字体的名称（可以自定义）
          font-family：wal520；
            e/引用字体文件 （固定写法）
                src：url("LiDeBiao-Xing3.eot? #iefix") format（"svg"） //兼容ie678
    }            
}

字体图标

1.字体约束文件里有自己的编码格式
2.在html当中要使用一段文字符表示一个图标的是 字体文件的编码
3.写在content当中，字符编码保持一致
4.转移符+对应的字符
span：：befor{
    font-family：iconfont；
    content："\e911";
    font-size:52px;(字体具有的功能 图标也具有)
}
