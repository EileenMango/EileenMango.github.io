window 的三个属性 location navigator history

Location : 地址的意思 （属性和方法）

属性 （和服务器有关）

  • location.href ; 地址，页面
  • location.hash ; 返回url中#后面的内容，包含#
  • location.host ; 主机名，包括端口
  • location.hostname ; 主机名
  • window.location.pathname ; url中的路径部分
  • window.location.protocal ; 协议 一般是http，https
  • window.location.search ; 查询字符串

方法

  • location.assign 改变浏览的地址，就是页面跳转
  • location.href
  • location.replace() 页面跳转 不会有历史记录
  • location.reload（） 页面刷新

navigatot 与浏览器，系统有关 （属性）

  1. userAgent (浏览器)
  2. platform (浏览器支持的系统，win/mac/linux)
