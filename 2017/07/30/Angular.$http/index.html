$http 服务

说明： angular 提供了 xhr 的功能，类似于 jquery 中的 $.ajax()

跨域请求的方式

  1. jsonp 为了防止全局污染，把 jsonp 的回调函数放在 angular.callbacks 对象中;
  2. window.name : 同一个标签中页面共享同一个 name 属性;
  3. iframe
  4. postMessage

注意： 在 angular 直接使用 $http.jsonp 请求时， 需要对 url 地址做信任操作 通过调用 $sce 服务的 trustAsRescourceUrl()
