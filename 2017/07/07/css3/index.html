手册

  1. [] 表示全部可选项
  2. || 表示或者
  3. | 表示多选一
  4. ？ 表示0个或者1个 0 || 1
    • 表示0个或者多个 >0
  6. {} 表示范围

选择器（3种）

属性选择器

  1. E[ attr ] 表示存在attr属性即可
  2. E[ attr=var ] 表示属性值完全等于val
  3. E[ attr*=val ] 表示的属性值里面包含val字符并且在 “任意” 位置
  4. E[ attr^=val ] 表示的属性值里面包含了val字符并且在 “开始” 位置
  5. E[ attr$=val ] 表示的属性值里面包含了val字符并且在 “结束” 位置
  6. E[ attr ][ href=val ]

伪类选择器

css3之前的伪类选择器 ：link，：active，：visited， ：hover

css3 新增选择器

序选择器/结构选择器：

步骤：

  1. 找到li元素
  2. 通过li找到父元素
  3. 通过父元素找到所有的子元素 child
  4. 再去找第几个子元素 first last nth
  5. 确定是不是li 如果是就选中 如果不是那么就是无效选择器

1/ li:nth-child(2) 选择到第二个li

2/ li:nth-child(2n) n的取值范围：正整数 0 —— 无穷大 选择到第2列的所有li

3/ li:nth-child(2n-1) n这个变量必须要在公式前面 选择到第一列的所有的li

4/ li:nth-last-child(2) 选择倒数第二个li

5/ li：nth-last-child(-n+2) 选择后面二个li

单个选择器 empty

span：empty {display：none}

锚点选择器 target

h1：targer {color：red} 当前锚点对应的元素会被选中

排除选择器 not

.class:not(#id) 排除ID为 id的这个元素

伪元素选择器:: (伪类：)

  1. ::before $$ ::after **
  2. ::first-letter 选中li当中的第一个字或者字母 只有块级元素才能使用
    li::first-letter {font-size:20px; line-height:30px}
  3. first-line 文本第一个行
  4. p:nth-child（2）：：selection{background：pink} 内容被选中后的变化

伪类和伪元素的区别

伪类的效果可以通过添加一个实际的类来达到，而伪元素的效果则需要通过添加一个实际的元素才能达到，这也是为什么称伪类: 和 伪元素:: 的原因

颜色

  1. opacity 子元素会继承父盒子的透明度 里面的文字也透明
  2. transparent 设置完全透明的颜色 子元素不会被继承
  3. rgba 透明度也不会被继承

文本阴影

text-shadow （多个阴影用逗号分隔）
第一个参数:x轴方向的偏移
第二个参数:y轴方向的偏移
第三个参数:阴影的模糊度 大于0
第四个参数:阴影的颜色

盒模型

box-sizing：content-box; 从内容开始计算 （标准模式盒模型）
box-sizing: border-box; 从边框开始计算 (怪异模式盒模型)

边框

圆：
border-radius：50%

盒子阴影：

  1. 外阴影 box.shadow:10px 10px 10px #ccc
  2. 内阴影 box.shadow:10px 10px 10px #ccc inset;
  3. 阴影延伸 box.shadow :0 0 10px 10px #ccc inset ;(第4个参数是阴影的延伸)

边框图片：

  1. 资源地址：border-image-source：url(“image/xxx”)
  2. 裁剪的尺寸:border-image-slice:27 27 27 27 裁剪的尺寸
  3. 边框的宽度：border-image-width：10px 决定图片显示的大小 图标会自适应显示在范围内
  4. 平铺的方式（3种）border-image-repeat：round/repeat/stretch

背景

背景-尺寸

  1. 让背景图和外容器一样大小

background-size：600px 400px；

  1. 让背景图自适应完全显示在盒子里 等比缩放

background-size：100% auto;

  1. 让背景图片，完全显示在盒子里面

background-size：contain;

  1. 让背景图片完全铺满整个盒子

background-size：cover;

背景-原点

1.默认的背景原点是

background-origin：padding-box；

2.让背景图的原点从内容开始铺

background-origin：content-box；

3.让背景图的原点从边框开始铺

background-origin：border-box

背景-裁剪

1.默认的裁剪方式 边框以外的都被裁剪掉

background-clip：border-box

2.边距以外的都被裁剪掉

background-clip:padding-box

3.内容以外的都被裁剪掉

background-clip：content-box

背景-多背景

background：

url（”images/xx”）left top;

url（”images/xx”）right top;

url（”images/xx”）right buttom;

url（”images/xx”）left buttom;

渐变

线性渐变

linear-gradient(45deg,yellow.blue)

参数：渐变的方向，起始的颜色，结束的颜色

径向渐变

radial-gradient(150px ,at center,blue,yellow)

参数：放射半径，中心原点，起始颜色，结束颜色

过渡

transition 元素属性发生改变时触发平滑过渡 （补间动画）

组合写法：transition：all 4s ease 1s

拆分：

1.需要过渡的属性 transition-property：all（border-color，width，hegith……）

2.过渡的时长 transition-duration：4s；

3.动画的速度 transition-timing-function：ease（先快后慢）/ linear(匀速的)……

4.过渡的延时 transition-delay：1s；
